This folder holds parcel shapes, parcel assessor's data, and zoning data for Boston, which can all be found at data.boston.gov.

parcel data (2025):
https://data.boston.gov/dataset/parcels-2025

property assessment data (FY2025):
https://data.boston.gov/dataset/property-assessment

zoning data (2025-03-24 release):
https://data.boston.gov/dataset/boston-zoning-subdistricts
